# FrameLand

## Grid Layer
- Ground Bottom     地面最底层
- Ground Middle     地面中间层
- Ground Top        地面最顶层
- Stuff 1           地面装饰层
- Stuff 2           地面装饰层
- Dig               挖坑图层
- Water             浇水涂层
- Instance          角色涂层
- Front 1           遮挡层
- Front 2           遮挡层
- Collision         边缘碰撞层
- Grid Properties   网格的属性，用来标识行为可行区域
    - CanDig            可以挖土
    - CanDropItem       可以扔弃
    - CanPlaceFurniture 可以摆放家具
    - NPC Obstacle      NPC障碍物
